# Compare Two Numbers in C++

## 📌 Description
This program compares two numbers entered by the user and determines which number is greater or if both numbers are equal.

## 🚀 How It Works
1. User enters two numbers.
2. Program compares them using if-else if statements.
3. Displays:
   - A is greater
   - B is greater
   - Both are equal

## 📥 Input Example

```text
Enter 1st number:
20

Enter 2nd number:
15
```

## 📤 Output Example

```text
A is greater
```

## 🛠️ Concepts Used
- C++
- Variables
- User Input (cin)
- User Output (cout)
- If-Else If Statements
- Comparison Operators

## 🎯 Learning Outcome
- Comparing values
- Using conditional statements
- Taking multiple inputs from the user
